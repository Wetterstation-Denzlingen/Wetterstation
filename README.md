# Wetterstation
Eine Neue Version (erste Stelle, vor dem Punkt) entsteht immer dann, wenn ein neues Bauteil hinzugefügt wird. Eine neue Unterversion (zweite Stelle, nach dem Punkt) beinhaltet nur neue Funktionen, ohne, dass Bauteile hinzugefügt, entfernt oder geändert werden müssen.
Jede neue Version enthält nur neue Kommentare. Bei nicht kommentierten Befehlen können vorherige Versionen zum besseren Verständnis beitragen.

A new version (first digit, before the dot) is always created when a new component is added. A new sub-version (second digit, after the dot) contains only new functions, without components having to be added, removed or changed.
Each new version contains only new comments. In the case of non-commented commands, previous versions may help to improve understanding.
